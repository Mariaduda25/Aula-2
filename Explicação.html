Segue explicação sobre as Tags

HTML

Esse elemento representa a raiz de um HTML ou XHTML documento. Todos os outros elementos devem ser descendentes desse elemento.

HEAD

Esse elemento providência informações gerais sobre o documento, incluidno títulos e links.

BODY

Esse elemento representa o conteúdo de um documento HTML.

HEADER

Esse elemento representa um grupo de suporte introdutório ou navegacional.

SECTION

Esse elemento representa uma seção genérica contida em um documento HTML.

DIV

Esse elemento é de divisão para conteúdo de fluxo. Pode ser utilizado para agrupar elementos para fins de estilo.

FOOTER

Esse elemento representa um rodapé para o conteúdo de seção mais próximo.

H1 ao H6

Esses elementos representam seis níveis de título de seção

H7

Esse elemento é inválido, o limite que ele consegue diminuir a letra é o H6.

HR

Esse elemento representa uma quebra de temática entre elementos de nível de parágrafo.

P

Esse elemento representa um parágrafo.

UL

Esse elemento representa uma lista de itens sem ordem rígida.

LI

Esse elemento é usado para representar um item que faz parte de uma lista.

BR

Esse elemento produz uma quebra de linha em um texto.

NAV

Esse elemento representa uma seção de uma página que aponta para outras páginas ou para outras áreas da página.

DD

Esse elemento fornece detalhes ou uma definição mais completa do termo precedente.

DL

Esse elemento engloba uma lista de pares de termos e descrições.

ADDRESS

Esse elemento fornece informações de contato para seu ancestral

ou mais próximo; no segundo caso, ele se aplica ao documento inteiro.

ASIDE

Esse elemento representa uma seção de uma página que consiste de conteúdo que é tangencialmente relacionado ao conteúdo do seu entorno, que poderia ser considerado separado do conteúdo.

CODE

Esse elemento apresenta seu conteúdo estilizado de maneira a indicar que o texto é um pequeno fragmento de código.

CITE

Esse elemento representa uma referência a um trabalho artístico.

MARK

Esse elemento representa um trecho de destaque em um texto.

S

Esse elemento renderiza um texto tachado ou uma linha que corta o texto.

IMG

Esse elemento representa a inserção de uma imagem no documento.
